23. 노드 리액트 기초 강의 #23 Proxy Server ?

Proxy Server
1. 아이피를 Proxy Server에서 임의로 바꿔 버릴수 있다.
그래서 인터넷에서는 접근하는 사람의 IP를 모르게 된다.
2. 보내는 데이터도 임의로 바꿀수 있다.
3. 방화벽 기능
4. 웹 필터 기능
5. 캐쉬 데이터, 공유 데이터 제공 기능

Proxy Server 사용 이유!
1. 회사에서 직원들이나 집안에서 아이들 인터넷 사용 제어
2. 캐쉬를 이용해 더 빠른 이터넷 이용 제공
3. 더 나은 보안 제공
4. 이용 제한된 사이트 접근 가능
